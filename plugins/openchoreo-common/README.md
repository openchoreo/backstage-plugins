# @openchoreo/backstage-plugin-common

Common package for shared types, constants, and utilities across OpenChoreo Backstage plugins.

## Usage

```typescript
import { CHOREO_ANNOTATIONS, CHOREO_LABELS } from '@openchoreo/backstage-plugin-common';
```

## Contents

- **CHOREO_ANNOTATIONS**: Shared annotation constants used across frontend and backend plugins
- **CHOREO_LABELS**: Shared label constants
